# 🎾 Ball Animation using HTML & CSS

A simple and responsive ball animation created using **HTML** and **CSS**. This project demonstrates how CSS animations and keyframes can be used to simulate realistic ball movement, including bouncing and smooth transitions, without using JavaScript.

## ✨ Features

* Pure HTML and CSS implementation
* Smooth bouncing ball animation
* CSS `@keyframes` for motion effects
* Lightweight and beginner-friendly code
* Easy to customize animation speed, size, and colors
* Responsive design that works across different screen sizes

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations, Keyframes, Transform, Timing Functions)

## 📚 Purpose

This project was built to practice CSS animation concepts such as `@keyframes`, `transform`, `animation`, and timing functions. It serves as a great beginner project for understanding how to create engaging visual effects using only HTML and CSS.

Feel free to explore, modify, and use this project as a learning resource or as a starting point for more advanced CSS animations.
